# Datasets

Reference data tables for the SBPR Valheim mods — structured data that both
humans and tooling read. Unlike the prose under `../design/` and `../v0.1.0/`,
these documents are meant to be looked *up*, not read top to bottom.

## Contents

- **[`PIECES_AND_CRAFTABLES.md`](PIECES_AND_CRAFTABLES.md)** — the master table of
  every buildable piece and craftable item: recipes, costs, crafting stations,
  and progression tier. The reference when reconciling code against intended
  recipe values (and what the spec-drift watchdog checks against).

See [`index.md`](index.md) for the machine-readable manifest.
