# docs/v2/investigations — README

De-risking **spikes** for the v2 (Black Forest) cartography tier: throwaway
proofs that retire a specific unknown *before* the impl cards commit to it. Each
file is a point-in-time investigation with a `verdict` (GO / GO-WITH-CAVEATS /
BLOCKED) in its frontmatter — read it as "what we proved and what's still open,"
not as a living spec. The locked build target stays in
[`../planning/requirements.md`](../planning/requirements.md); investigations feed
*how* those requirements get spec'd into impl cards.

See [`index.md`](index.md) for the manifest.
