# docs/v2 — Trailborne v2 (Black Forest tier)

Version-scoped docs for the **Black Forest** content tier of SBPR Trailborne.

The v2 headline is the **cartography system**: the **Surveyor's Table**, **Local
Maps**, and the **Cartographer's Kit**. The durable design intent + decision log
lives in [`../design/cartography-v2.md`](../design/cartography-v2.md); this tree is
where ratified decisions get promoted into a structured, locked requirements spec
(mirroring how `docs/v0.1.0/planning/` holds the Meadows-tier spec).

## Read order

1. [`../design/cartography-v2.md`](../design/cartography-v2.md) — the full design
   doc + decision history (why each thing is the way it is).
2. [`planning/requirements.md`](planning/requirements.md) — the locked v2 requirements
   distilled from the ratified decisions (what to build).

## Status

v2 is **specced, not started**. The Meadows-tier loop (v0.1.0 / the 0.2.x line)
is the active build; v2 begins once it's locked. See
[`../v0.1.0/MILESTONES.md`](../v0.1.0/MILESTONES.md).
