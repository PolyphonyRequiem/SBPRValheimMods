# v1 — working docs

Version-scoped working docs for the **v1** line of Trailborne (the
vertical-slice architecture and the fixes layered on top of the v0.1.0 spec).

Durable design intent lives in [`../design/`](../design/); the locked v0.1.0
spec lives in [`../v0.1.0/`](../v0.1.0/). This tree holds the architecture plan
and research notes specific to the v1 implementation line.

## Subdirectories

- [`architecture/`](architecture/) — the vertical-slice `Features/` plan and
  structural decisions for the v1 codebase.
- [`research/`](research/) — implementation investigations and fix notes
  (point-in-time records of how a specific problem was solved).

See [`index.md`](index.md) for the machine-readable manifest.
