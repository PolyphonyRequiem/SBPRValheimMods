# v0.1.0

Version-scoped working documentation for the **Trailborne v0.1.0** release —
the first SBPR mod to ship. Everything here is anchored to this specific version;
the next release gets a `../v0.2.0/` sibling rather than editing these in place.

For the durable *why* behind the design, see [`../design/`](../design/). This
folder is the *what* and *how* for v0.1.0 specifically.

## Start here

1. **[`MILESTONES.md`](MILESTONES.md)** — the milestone map (M0→M4) and current
   status. The fastest way to see what v0.1.0 covers.
2. **[`planning/requirements.md`](planning/requirements.md)** — the full
   requirements spec. The source of truth for what each piece must do; re-read
   before coding to avoid drift.

## Planning

The [`planning/`](planning/) subfolder holds the artifacts that preceded code:

- **[`planning/initialization.md`](planning/initialization.md)** — the raw idea
  verbatim from Daniel plus the doctrine carried in from prior context.
- **[`planning/requirements.md`](planning/requirements.md)** — requirements
  broken out of that raw idea into a buildable spec.
